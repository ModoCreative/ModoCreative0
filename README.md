# MoDo Creative Website

## Quick Start in VS Code

1. Open the `modo-creative-website` folder in VS Code.
2. Install [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension.
3. Right-click `index.html` → **Open with Live Server**.
4. Site opens in browser at `http://127.0.0.1:5500/index.html`.

## Edit & Customize
- **Branding**: Edit colors in `css/style.css` (variables).
- **WhatsApp**: Update phone in HTML (`data-phone="201559233383"`).
- **Images**: Replace SVG placeholders in `assets/images/`.
- **Pages**: All 5 pages fully responsive.
